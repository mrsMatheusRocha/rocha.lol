Featured Project

Assassin's Quiz

  • React
  • Next.js
  • API
  • Lottie

Assassin's Quiz features a dark, Assassin's Creed-inspired design with a central “Play” button for randomized lore quizzes and a sidebar of community-submitted challenges. Questions appear on responsive cards powered by Flexbox and CSS Grid, with animated answer buttons that signal right or wrong. A progress bar tracks your quiz journey, ending in a score summary and shareable link. Optimized images, custom fonts, keyboard navigation, ARIA labels, and lazy loading ensure both performance and accessibility.
